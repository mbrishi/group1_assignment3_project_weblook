Welcome to NYC Motor Vehicle Collisions Dashboard

This website provides insights into motor vehicle collisions across New York City. It leverages publicly available data to support the analysis of collision frequency, contributing factors, and their impact on public safetyS.

Project Purpose:

The primary objective of this platform is to make crash data easy to understand. By providing visual insights, the dashboard helps people, city planners, and safety experts make better decisions and promote safer streets.

Key Features:

  • Interactive data visualizations developed using using Chart.js
  • Insights into collisions categorized by borough, injury severity, and contributing causes
  • Buttons that simulate adding, viewing, updating, and deleting collision data
